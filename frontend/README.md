# Expertise Auto - Frontend Professionnel

Application web moderne de gestion d'expertise automobile.

## Installation

```bash
npm install
```

## Configuration

Créer un fichier `.env` :

```
VITE_API_URL=https://votre-backend.up.railway.app/api
```

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Déploiement Vercel

```bash
vercel
```

Variables d'environnement à configurer sur Vercel :
- `VITE_API_URL` = URL de votre backend Railway

## Fonctionnalités

✅ Authentification JWT
✅ Dashboard avec statistiques
✅ Gestion des bureaux (CRUD complet)
✅ Gestion des rapports d'expertise
✅ Recherche par sinistre
✅ Interface responsive
✅ Design moderne (Tailwind CSS)
